# CarusoGianlucaLB-259
## Datensatzbeschreibung

Der verwendete Datensatz enthält Informationen über Studierende und deren Studienverlauf. Jede Zeile steht für eine Person. Die Spalten beschreiben verschiedene Merkmale wie Alter, Geschlecht, Einkommen der Familie, Internetzugang, Lernzeit, Anwesenheit, Noten, Semester, Fachrichtung und Bildungsstand der Eltern. Das Zielfeld ist Dropout. Es zeigt, ob eine Person das Studium abgebrochen hat oder nicht. Der Datensatz eignet sich für Machine Learning, weil verschiedene Merkmale miteinander zusammenhängen und daraus Vorhersagen zum Studienabbruch erstellt werden können.

## Datenschutz und Schutzmassnahmen

Im Datensatz sind keine direkten persönlichen Informationen wie Namen, Adressen, Telefonnummern oder E-Mail-Adressen enthalten. Die Personen können deshalb nicht direkt identifiziert werden. Für das Projekt wurden fehlende Werte entfernt und die bereinigten Daten lokal gespeichert. Die Daten werden nur für Analysezwecke verwendet und nicht weitergegeben. Trotzdem enthalten einzelne Spalten, zum Beispiel Einkommen oder Bildungsstand der Eltern, sensible Informationen. Deshalb wird der Datensatz nur anonymisiert verwendet und nicht mit anderen personenbezogenen Daten verbunden.
